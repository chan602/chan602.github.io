The Origin Story

For years I tracked daily goals using handwritten lists, then experimented with ChatGPT as a task manager — a workflow that sounded promising but turned out to be fundamentally mismatched. AI is great at reasoning; it's not reliable for deterministic state. Tasks were misinterpreted, dropped, or silently lost.

That frustration led me to build a hybrid system: structured Firebase-backed task logic paired with the Anthropic API for evaluation. The result lives on this site as Three Wins — a personal tracker for my daily physical, mental, and spiritual goals.

Now I'm scaling it. threedailywins.com is the multi-user app version — same framework, built for friends and eventually anyone who wants a structured, intentional daily routine. This blog tracks the build from idea through launch.
