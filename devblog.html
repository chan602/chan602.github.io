The Origin Story

I built a structured, Firebase backed task logic paired with the Anthropic API for AI evaluation of my todo lists. The result lives on this site as Three Wins, a personal tracker for my daily physical, mental, and spiritual goals.

Now I'm working on scaling it. threedailywins.com is a multi-user app version with the same framework, built for friends and eventually anyone who wants to log and track their 3 win days. This blog tracks the build from idea through launch. Thank you for reading!
