# chan602.com

Personal website and portfolio.

## Overview

This website serves as a central hub for my work, projects, and experiences across science, healthcare, and technology. It is designed to present both my academic background and the systems I build, with a focus on clarity, simplicity, and functionality.

The site reflects my broader goal of combining scientific thinking with practical tools that improve how people learn, work, and live.

## Features

-  Personal and academic background
-  Research interests and experience
-  Project portfolio
-  Personal interests and activities
-  Media and experience gallery
-  External links and contact information

## Highlighted Projects

### 3 Wins
A full stack productivity app centered around achieving daily physical, mental, and spiritual wins.  
Built with React, Firebase, and AI evaluation (Anthropic API), with features including task tracking, streaks, and social elements.

### Other Work
Additional projects focus on:
- Frontend focused web development
- Practical tools for personal productivity
- Systems that integrate structured logic with AI capabilities

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Frameworks/Tools:** React (for project work), Vite
- **Backend/Services:** Firebase (Firestore, Auth)
- **Deployment:** Static hosting (GitHub Pages / my domain)

## Design Philosophy

- **Clarity over complexity:** minimal, readable interfaces
- **Function-driven design:** each component serves a clear purpose
- **Iterative development:** continuous refinement based on real use
- **Personal authenticity:** reflects both technical and non-technical aspects of my work

## Purpose

This site is intended to:
- Showcase my work to potential employers and collaborators
- Provide a structured overview of my academic and project experience
- Serve as a platform to share and iterate on ideas
